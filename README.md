Internal team workspace for building our own Hindi **vegetable-drama** YouTube channel — modelled on the proven viral format (anthropomorphic sabzi acting out love, betrayal/धोखा, rags-to-riches and karma).

## Production pipeline (summary)
1. Pick/write a story (keep it distinct — see Guardrails on the site).
2. Lock the character design sheet.
3. Generate each scene's still from its **Image Prompt** (text-to-image).
4. Animate each still with its **I2V Prompt** in Seedance / Veo3 / facelessdev.com.
5. Add Hindi narration + character voices.
6. Stitch in order, add BGM/SFX/captions, honour the transitions.
7. Title + thumbnail per the formula. Upload daily.
